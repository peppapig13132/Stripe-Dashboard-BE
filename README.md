- [x] Free to paid subscriptions
- [x] Free trials
- [ ] Annual Run Rate


## Calculation

### Customer Churn Rate

```
Churn Rate = (Number of Canceled Subscriptions / Number of Active Subscriptions at the Start) * 100
```
